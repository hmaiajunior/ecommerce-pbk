# Playbekids Website UI Kit

Recreates the **Playbekids e-commerce website** — a Brazilian children's fashion store (boys' clothing, atacado + varejo).

## Stack
- React + Babel (inline JSX)
- Nunito font (Google Fonts)
- CSS custom properties from `../../colors_and_type.css`

## Screens covered in index.html
1. **Homepage** — Hero banner + product grid + promotional strip
2. **Product Detail** — Product image, sizes, price, CTA
3. **Atacado (Wholesale)** — Info page for resellers

## Files
- `index.html` — Full interactive UI
- `Header.jsx` — Top nav with logo, menu, cart
- `Hero.jsx` — Full-width hero banner with blobs
- `ProductCard.jsx` — Product card component
- `Footer.jsx` — Site footer

## Notes
- No live website was accessible; UI is inferred from logo + social post design language
- Product imagery uses placeholders (colored boxes with emoji)
